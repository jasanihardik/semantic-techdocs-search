Title: Security Best Practices

Summary
Follow these security best practices to keep company data safe.

Key practices:
- Lock your screen when away.
- Use strong, unique passwords and enable MFA.
- Do not use personal accounts for company work.
- Report suspicious emails (phishing) to security@company.example.

Password policies & recovery:
- See Password Recovery Process for resetting account credentials.
- Use company password manager to store secrets.

Remote work implications:
- When working from home, ensure your Wi-Fi is encrypted and devices are updated.
- Use the VPN when accessing internal services; see VPN Access and Setup.
